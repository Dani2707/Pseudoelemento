Pseudo-elementos

::After

En CSS, ::after crea un pseudo-elemento que es el último hijo del elemento seleccionado. Es comúnmente usado para agregar contenido cosmético a un elemento con la propiedad content. Es en linea (inline) de forma predeterminada,se agrega despues sin necesidad de editar el html

::Spelling-error

El pseudo-elemento CSS ::spelling-error representa un segmento de texto que el agente de usuario (en-US) ha marcado como eliminado incorrectamente.

Tecnologia experimental y algunos no lo soportan
